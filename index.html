7 Langkah Install Macro Moonwalk

Tutorial Install Macro Moonwalk @M1XO

Pastikan Mengikuti Semua Dari Awal Agar Tidak Terjadi Kesalahaan
1
📥 Unduh AutoHotkey

Unduh dan instal AutoHotkey melalui:
👉 https://www.autohotkey.com/

🖥 Tampilan situs resmi AutoHotkey
💡 Tip: Pastikan Anda mendownload dari situs resmi untuk keamanan.
2
🔀 Pilih Versi AutoHotkey

Anda dapat menggunakan AutoHotkey Versi 1 maupun Versi 2 — keduanya didukung untuk menjalankan script macro.

Masih Ditampilan Original Website
ℹ️ Info: Auto Hotkey 100% Aman Dan Sudah Di Verifikasi Oleh Norton
3
🛡️ Nonaktifkan Antivirus Sementara

Nonaktifkan sementara Windows Defender atau antivirus apabila memblokir proses instalasi atau script.

🛡️ Windows Defender Security Center
4
✅ Langkah Selanjutnya

Setelah AutoHotkey Berhasil Terinstall Silahkan Tutup AutoHotkey.

Jika Tampilan Ini Muncul Silahkan Di Close Aj
5
📂 Jalankan Script

Setelah pembayaran dikonfirmasi, Download atau simpan file script macro yang telah diberikan. Lalu Double Click Untuk Membuka File

📁 File Macro Moonwalk
💡 Tip: Simpan file script di folder yang mudah diingat, misalnya C:\Macro\Violence District @Mixo.ahk
6
🔐 Pop Up Saat Install

Jika muncul Notifikasi Seperti Di Foto klik Yes untuk melanjutkan. Tunggu Beberapa Detik Lalu Klik OK

Notifikasi
⚠️ Penting: Klik Yes agar script dapat berjalan
7
🚀 Jalankan Script & Siap Digunakan

Jalankan file script, lalu macro siap digunakan!

🎯 Script macro siap dijalankan
✅ Selesai! Macro Anda sekarang siap digunakan. Selamat mencoba! 🎉
